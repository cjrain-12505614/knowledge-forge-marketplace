# knowledge-forge-marketplace

knowledge-forge 플러그인 배포용 Claude Code 마켓플레이스.

## 설치

```bash
claude plugin marketplace add cjrain-12505614/knowledge-forge-marketplace
claude plugin install knowledge-forge@knowledge-forge-marketplace
```

## 배포

소스 레포(`knowledge-forge`)에서 `build.sh --deploy` 실행 시 이 레포로 자동 배포됩니다.
