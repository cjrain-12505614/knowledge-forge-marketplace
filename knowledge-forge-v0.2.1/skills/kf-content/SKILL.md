# KF-Content — SNS 콘텐츠 제작

일일 브리핑을 기반으로 X, 페이스북, 인스타그램 각 플랫폼에 최적화된 콘텐츠를 제작한다.

## 플랫폼별 전략

### X (Twitter)

**파일**: `archive/YYYY-MM-DD/content/x.md`

- **분량**: 280자 이내 (한글 기준 140자)
- **형식**: 핵심 팩트 1개 + 짧은 인사이트 + 해시태그
- **톤**: 임팩트 있고 간결. 전문가가 쓴 속보 느낌.
- **구성**: 헤드라인 3건 각각 1개 트윗 + 스레드용 요약 1개 = 총 4개 초안
- **해시태그**: #AI #인공지능 + 관련 키워드 2~3개. 최대 5개.

작성 예시:
```
Claude 4 출시 — 코딩 벤치마크 SWE-bench에서 72.5% 달성, 전작 대비 15%p 향상.
에이전틱 코딩의 새 기준선이 될 전망.

#Claude4 #Anthropic #AI #에이전틱코딩
```

### Facebook

**파일**: `archive/YYYY-MM-DD/content/facebook.md`

- **분량**: 500~800자 (한글 기준)
- **형식**: 도입(훅) + 본문(상세 분석) + 마무리(인사이트/질문)
- **톤**: 전문적이되 대화체. "~입니다" 보다 "~이에요" / "~하네요" 선호.
- **구성**: 1개 포스트로 당일 핵심 2~3건 통합 정리
- **CTA**: 마지막에 독자 참여 유도 질문 포함

### Instagram

**파일**: `archive/YYYY-MM-DD/content/instagram.md`

- **분량**: 카드 5~7장 분량의 텍스트
- **형식**: 카드뉴스 스크립트 (슬라이드 번호 + 텍스트)
- **톤**: 시각적, 핵심만, 키워드 중심
- **구성**:
  - 슬라이드 1: 타이틀 (관심 끄는 질문 또는 선언)
  - 슬라이드 2~5: 핵심 내용 (카드당 1가지 포인트)
  - 슬라이드 6: 요약 또는 인사이트
  - 슬라이드 7: CTA + 팔로우 유도
- **캡션**: 요약 + 해시태그 15~20개

카드뉴스 스크립트 형식:
```markdown
## 슬라이드 1
**[타이틀]**
오늘 AI 업계에 무슨 일이?

## 슬라이드 2
**[포인트 1]**
{핵심 내용 2~3줄}

...
```

## 공통 원칙

- **출처 표기**: 원문 링크는 Facebook에서만 본문에 포함. X는 리플라이로, Instagram은 "링크는 바이오에" 안내.
- **중립성**: 특정 기업 편향 없이 사실 기반으로 작성.
- **실무 관점**: 단순 뉴스 전달이 아닌, "이게 왜 중요한지"를 항상 포함.
- **더존비즈온 내부용이 아님**: SNS는 외부 공개 콘텐츠. 사내 정보는 절대 포함하지 않는다.

## 핵심 규칙

- 3개 플랫폼 콘텐츠를 **동시에** 생성한다.
- 같은 소재를 쓰되, 플랫폼 특성에 맞게 **완전히 다른 글**로 재작성한다.
- 헤드라인 Top 3 중 가장 임팩트 있는 것을 중심 소재로 선택한다.
