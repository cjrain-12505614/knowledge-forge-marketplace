# KF-Report — 주기별 트렌드 리포트

누적된 아카이브 데이터를 집계하여 주간/월간/분기별 트렌드 분석 리포트를 생성한다.

## 리포트 유형

| 유형 | 주기 | 트리거 | 파일 위치 |
|------|------|--------|-----------|
| **주간** | 매주 일요일 | 수동 또는 자동 | `archive/reports/weekly/YYYY-WNN.md` |
| **월간** | 매월 말일 | 수동 또는 자동 | `archive/reports/monthly/YYYY-MM.md` |
| **분기** | 분기 말 | 수동 또는 자동 | `archive/reports/quarterly/YYYY-QN.md` |

## 리포트 생성 절차

### 1단계: 데이터 수집 범위 확정

- 주간: 해당 주 월~일요일 (7일)
- 월간: 해당 월 1일~말일
- 분기: 해당 분기 3개월

### 2단계: 엔트리 집계

해당 기간의 모든 `archive/YYYY-MM-DD/entries/` 파일을 읽고 집계한다.

```markdown
## 수집 통계
- 총 엔트리: N건
- 카테고리별 분포:
  | 카테고리 | 건수 | 비율 | 전 기간 대비 |
  |----------|------|------|-------------|
  | C1 모델 발표 | X건 | X% | +N% / -N% |
  ...
- 주요 언급 기업 Top 10
- 주요 키워드 Top 20
```

### 3단계: 트렌드 분석

기간 내 테마 클러스터, 기업 동향, 기술 흐름을 분석한다.

#### 3-1. 핵심 트렌드 (3~5개)

각 트렌드에 대해:

```markdown
### 트렌드 1: {제목}

**요약**: 1~2문장 핵심.

**전개 과정**:
- MM/DD: {이벤트 1} (C1-01)
- MM/DD: {이벤트 2} (C3-05)
- MM/DD: {이벤트 3} (C7-02)

**주요 기업**: A, B, C

**향후 전망**: 다음 기간에 예상되는 전개 방향.

**더존비즈온 시사점**: 실무 적용 관점.
```

#### 3-2. 기업 동향 분석

주요 기업별 기간 내 활동을 요약한다. `archive/timelines/companies/` 파일을 참조한다.

```markdown
### 기업별 주요 활동

| 기업 | 주요 이벤트 | 전략 방향 |
|------|-------------|-----------|
| OpenAI | GPT-5 발표, API 가격 인하 | 모델 성능 + 가격 경쟁력 |
| Anthropic | Claude 4 출시, MCP 생태계 | 안전성 + 개발자 경험 |
```

#### 3-3. 기술 흐름 맵

기간 내 기술 키워드의 빈도 변화와 상호 관계를 분석한다.

```markdown
### 기술 키워드 트렌드

**상승 키워드**: agent, MCP, local AI, on-device
**하강 키워드**: prompt engineering, fine-tuning
**신규 등장**: {이전 기간에 없던 키워드}
**지속 강세**: multimodal, reasoning, open-source
```

### 4단계: 비교 분석

이전 동일 기간과 비교한다 (월간이면 전월, 분기면 전분기).

```markdown
## 전 기간 대비 변화
- 수집 건수: N건 → M건 (±X%)
- 가장 성장한 카테고리: CX (+Y%)
- 가장 줄어든 카테고리: CX (-Y%)
- 새로 등장한 기업: {목록}
- 사라진 키워드: {목록}
```

### 5단계: 더존비즈온 전략 시사점

PM/개발 실무에 직접 영향을 줄 수 있는 트렌드를 정리한다.

```markdown
## 더존비즈온 시사점

### 즉시 검토 필요
- {구체적 액션 아이템 1}
- {구체적 액션 아이템 2}

### 중기 관찰 (1~3개월)
- {관찰 포인트 1}
- {관찰 포인트 2}

### 장기 대비 (3개월+)
- {전략적 고려사항}
```

### 6단계: 리포트 저장

해당 유형의 디렉토리에 저장한다.

## 월간 리포트 필수 섹션

1. **이달의 헤드라인 5선** — 가장 임팩트 있었던 소식 5건
2. **수집 통계** — 카테고리별 분포, 전월 대비 변화
3. **핵심 트렌드 (3~5개)** — 전개 과정 + 전망
4. **기업 동향** — 주요 기업별 활동 요약
5. **기술 키워드 트렌드** — 상승/하강/신규/지속
6. **전월 대비 변화** — 수치 비교
7. **더존비즈온 시사점** — 즉시/중기/장기 액션
8. **다음 달 관전 포인트** — 예고된 이벤트, 예상 트렌드

## 분기 리포트 추가 섹션

월간 섹션에 더해:

9. **분기 타임라인** — 주요 이벤트를 시간순으로 나열
10. **산업 구도 변화** — 경쟁 관계, 파트너십, M&A 흐름
11. **규제 환경 변화** — 정책·규제 동향 종합
12. **투자 흐름 분석** — 펀딩 규모, 분야별 집중도

## 핵심 규칙

- **데이터 기반**: 모든 주장에 엔트리 ID와 출처를 명시한다.
- **비교 필수**: 이전 기간과의 비교 없이 리포트를 완성하지 않는다 (첫 달은 예외).
- **액션 지향**: "이런 트렌드가 있다"로 끝내지 않고, 실무 대응 방향을 제시한다.
- **분량 가이드**: 주간 1,000~1,500자, 월간 3,000~5,000자, 분기 5,000~8,000자 (한글 기준).
